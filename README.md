## Development

Install dependencies:
```bash
pip install figpack numpy pandas matplotlib
figpack extensions install figpack_slides
npm install
```

Build:
```bash
python index.py
```

Watch mode (auto-rebuild):
```bash
./watch.sh
```

Serve locally (in separate terminal):
```bash
npx serve build
```
