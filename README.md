## Development

Install dependencies:

```bash
pip install figpack numpy pandas matplotlib
figpack extensions install figpack_slides
npm install
```

First build the presentation:

```bash
python index.py
```

Then serve it locally:

```bash
npx serve build
```

Or in watch mode:

```bash
./dev.sh
```
